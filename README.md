# Programming Logic & Data Manipulation for Business Administration 📊🐍

This repository contains practical exercises designed to apply programming logic and data analysis to real-world administrative scenarios. The main focus is using Python and data science libraries to optimize business decision-making.

## 🎯 Objective
To practice solving management problems (such as sales performance, profit margins, and inventory control) using control structures, loops, and data manipulation tools.

## 📂 Project Content
The main notebook covers the following topics:

1.  **Logic and Conditionals:** * Daily sales performance classification (Sales Traffic Light).
    * Profit margin analysis for profitable products.
2.  **Iterative Commands (Loops):**
    * Weekly average sales calculation.
    * Inventory and returns counting and accumulation.
3.  **Pandas and File Handling:**
    * Reading Excel reports (`.xlsx`).
    * Filtering specific data (e.g., sales above 30 units).
4.  **Statistical and Financial Calculations:**
    * Creating columns for Total Revenue and Average Ticket calculations.
5.  **Data Visualization:**
    * Bar charts to compare revenue by product using `Seaborn`.
    * Line charts for sales trends over time.

## 🛠️ Technologies Used
* **Python 3**
* **Pandas:** Data manipulation and analysis.
* **Seaborn / Matplotlib:** Statistical data visualization.
* **Jupyter Notebook / Google Colab**
